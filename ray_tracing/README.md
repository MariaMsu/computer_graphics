## how to run

```
$ mkdir build && cd build
$ cmake ..
$ make
& ./ray_tracing  # or ./ray_tracing -w
>> a picture with the result will appear in this directory
```

* http://www.cs.cornell.edu/courses/cs4620/2013fa/lectures/03raytracing1.pdf
